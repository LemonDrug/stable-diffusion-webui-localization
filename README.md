# stable-diffusion-webui-localization
Support multi-language localization display

支持中日韩英的语言本地化显示，请确保已经安装了谷歌浏览器，可以正常打开127.0.0.1:7860后，再使用本软件。

Support Chinese, Japanese, Korean and English language localization display, please make sure that Google Chrome has been installed, and you can use this software after you can open 127.0.0.1:7860 normally.

中国語、日本語、韓国語、英語のローカリゼーション表示をサポートします。Google Chrome がインストールされていることを確認してください。127.0.0.1:7860 を正常に開くことができたら、このソフトウェアを使用できます。

중국어, 일본어, 한국어 및 영어 현지화 표시를 지원합니다. Google 크롬이 설치되어 있는지 확인하고 127.0.0.1:7860을 정상적으로 열 수 있어야 이 소프트웨어를 사용할 수 있습니다.
